# Random Password Generator 🔐

This is a simple Python project that generates a secure random password using letters, numbers, and symbols.

## 📌 Features
- Generates strong random passwords
- User can choose password length
- Uses Python built-in libraries
- Beginner-friendly project

## 🛠 Technologies Used
- Python
- Random module
- String module

## ▶️ How to Run the Project

1. Install Python on your computer
2. Download or clone this repository
3. Open terminal in the project folder
4. Run the command:

python pass_gen.py

Generated password: A7@kP!2xQ9
